IntelliJ 단축키

IntelliJ 15개 주요 단축키 영상
Alt + Enter 코드에서 오류가 발견된 경우 해당 오류 위에 커서를 놓고 Alt Enter를 누르면 문제에 대한 수정 제안 목록이 표시
F2 에디터에서 오류과 경고 사이를 이동할 때 마우스를 사용하고 싶지 않다면, F2를 사용하여 다음 오류,경고 또는 제안으로 점프
Alt + 1 이 단축키를 사용하면 프로젝트 창이 열리고 거기에 포커스가 위치
Ctrl Ctrl 어디에서든 Ctrl 키를 두 번 눌러서 어떤 항목이든 실행. IDE안에 있는지의 여부나 열린 파일과 상관없이 Ctrl을 두 번 누르면 Run Anyting창이 열림 기본적으로 이 창에는 최근 실행 구성 목록이 표시. 하지만, 항목의 이름을 입력하여 다른 실행 구성을 검색하고 실행가능
Esc 에디터에 다시 포커스가 위치하게 하려면 Esc. 열린 도구 창과 상관없이 에디터에 다시 포커스가 위치하게 되어 코드 작업을 계속 수행가능
Ctrl + E Ctrl + E를 사용해 최근 실행했던 파일을 확인. 방향 키를 사용해 이동할 수 있는 최근 파일 상자가 표시
Ctrl + B Ctrl + B를 사용하면 심볼의 선언으로 이동. 클래스 이름 위에서 누르면 해당 클래스 파일로 이동
Alt + F7 이 단축키를 사용하면 어떤 항목이 사용된 모든 곳이 표시. 이 인터페이스 이름위에서 단축키를 누르면 이 인터페이스를 구현하는 것이 필드 선언이든 필드 선언이든 클래스이든 상관없이 해당 인터페이스가 사용된 모든 곳이 검색 창에 표시.
Ctrl + W / Shift + Ctrl + W 근처의 코드 선택 영역을 확대하거나 축소가능. 선택 영역을 확대할 경우 IntelliJ IDEA가 확대된 영역에서 유효한 다음 식을 자동으로 선택. Ctrl+Shift+W를 사용하면 선택 영역이 다시 커서 위치로 축소
Ctrl + / 줄 위의 아무 곳에서나 Ctrl + / 를 누르면 해당 코드 줄이 줄 주석으로 처리된다. 이미 주석으로 처리된 경우 동일한 단축키를 누르면 해당 줄의 주석 처리가 제거
Shift + Ctrl + Enter 코딩할 때 가장 유용한 단축키 중 하나. 코딩을 할 때 현재 구문 완성을 자주 사용한다면 대부분의 경우 단순히 코드 끝에 세미콜론을 추가함. 하지만, 더 복잡한 코드에 대해서도 작동하는데, 예를 들어 "for"루프를 작성하는 중 이 단축키를 누르면 IDEA가 중괄호를 추가하고 블록 안에 커서를 배치. "if"문에서는 괄호 및 중괄호를 추가하고 역시 적당한 위치에 커서를 배치 IDEA가 문을 완성하기 위해 더 이상 코드를 추가할 필요가 없을 경우에도 이 단축키를 사용해 적당한 다음 위치에 커서를 배치함.
Ctrl + Alt + Lr 해당 프로젝트의 표준에 맞도록 현재 파일의 서식을 지정함. 파일에서 변경된 줄 또는 전체 파일의 서식을 지정하도록 설정
Shift + Ctrl + Alt + T 이 단축키를 사용해 전체에 액세스할 수 있음. 심볼 또는 선택 영역 위에서 이 단축키를 누르면 사용 가능한 리팩토링 옵션이 표시. 그러면, 방향키를 사용해 하나를 선택하여 입력하거나, 리팩토링 왼쪽에 있는 숫자를 사용해 선택가능
Shift + Ctrl + A 이 단축키를 사용해 IntelliJ IDEA에서 모든 액션 검색가능. 액션 검색을 통해 액션뿐만 아니라 설정도 검색할 수 있으므로, 여기서 바로 설정을 변경가능
Shift Shift Shift 키를 두 번 누르면 모든 항목을 검색할 수 있는 검색 상자가 열림. 검색할 항목을 입력하면 클래스, 파일, 심볼 및 액션에 대한 결과가 표시
