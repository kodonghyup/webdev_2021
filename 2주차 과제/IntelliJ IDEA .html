IntelliJ IDEA

IntelliJ IDEA는 JetBrains사에서 제작한 상용 자바 통합 개발 환경이다. 줄여서 IntelliJ 혹은 IDEA로도 불린다.
이클립스재단의 이클립스와 썬 마이크로시스템즈의 넷빈즈로 대표되는 무료 자바 통합개발환경에서 볼랜드의 제이빌더(JBuilder)와 함께 얼마 안 되는 상용 개발 도구 가운데 하나이다.

IntelliJ IDEA 학생 라이센스 혜택

  1. 쿠폰 코드로 발급되는 학생 개인용 라이선스.
  2. 단일 JetBrains IDE에 유효.
  3. 최대 6개월 무료 구독.
  4. 향후 라이선스 갱신 시 25% 할인
  5. 해당 과정에 등록한 학생들과만 공유 가능.

IntelliJ IDEA 학생 라이센스 받는 방법

  1. JetBrains 사이트 접속
    www.jetbrains.com/community/education/#students
  2. 정보 입력 및 신청
    3. 학생 인증을 위한 학교 이메일 계정을 입력하고 확인
  3. 학교 이메일 확인
    4. 학교 이메일 주소로 로그인하여 인증메일 확인하고 "Comfirm Request"클릭 후 "JetBrains Account"사이트 창이 열리고, 맨 아래에 있는 "I Accept"클릭.
  4. 라이센스 발급 확인
    5. 이후 로그인하여 확인하면 학생 인증이 완료, 라이센스가 발급된 것을 확인

IntelliJ IDEA 설치방법

  1. 다운로드 사이트 접속
    https://www.jetbrains.com/idea/download/#section=windows
  2. Ultimate버전 다운로드하여 설치
    3. Zip 파일로 다운로드하여 압축 해제도 가능
  3. 최종 실행 및 인증
    4. 설치 또는 압축해제하여 IntelliJ를 실행하고 최초 인증 계정 정보만 입력하면 인증완료.
